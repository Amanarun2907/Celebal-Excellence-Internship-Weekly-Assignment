# Celebal Excellence Internship - Data Science Weekly Assignments

This repository contains all weekly assignments completed during the **Celebal Excellence Internship** in the **Data Science domain**.

## 📚 Program Overview

The internship spans 8 weeks, covering fundamental to advanced topics in Data Science, Machine Learning, Deep Learning, and AI. Each week focuses on a specific domain with hands-on assignments to reinforce learning.

## 📅 Weekly Assignment Breakdown

### Week 1: Python
- **Topics Covered:** Python fundamentals, data structures, functions, and libraries
- **Assignment:** `week1_assignment.ipynb`

### Week 2: Classical Machine Learning
- **Topics Covered:** Linear regression, decision trees, ensemble methods, and traditional ML algorithms
- **Status:** Coming soon

### Week 3: Classification
- **Topics Covered:** Classification algorithms, model evaluation, confusion matrix, and performance metrics
- **Status:** Coming soon

### Week 4: Introduction to Deep Learning
- **Topics Covered:** Neural networks, backpropagation, activation functions, and optimization techniques
- **Status:** Coming soon

### Week 5: RNN (Recurrent Neural Networks)
- **Topics Covered:** Sequential data processing, LSTM, GRU, and time series analysis
- **Status:** Coming soon

### Week 6: AE and GAN (Autoencoders and Generative Adversarial Networks)
- **Topics Covered:** Dimensionality reduction, generative models, image generation, and unsupervised learning
- **Status:** Coming soon

### Week 7: RAG and LLMs (Retrieval-Augmented Generation and Large Language Models)
- **Topics Covered:** Transformer architecture, prompt engineering, RAG systems, and LLM applications
- **Status:** Coming soon

### Week 8: Agentic AI
- **Topics Covered:** Autonomous agents, multi-agent systems, and AI decision-making frameworks
- **Status:** Coming soon

## 🚀 Getting Started

### Prerequisites
```bash
# Install Python 3.8 or higher
# Install required libraries
pip install numpy pandas scikit-learn matplotlib seaborn tensorflow jupyter
```

### Running the Assignments
```bash
# Clone the repository
git clone <repository-url>

# Navigate to the directory
cd Celebal-Excellence-Internship-Weekly-Assignment

# Launch Jupyter Notebook
jupyter notebook
```
