================================================================================
TROUBLESHOOTING
================================================================================

ISSUE 1: Package Installation Errors
-------------------------------------
Solution:
    pip install --upgrade pip
    pip install -r requirements.txt

ISSUE 2: Prophet Installation Failed (Windows)
-----------------------------------------------
Solution:
    pip install pystan
    pip install prophet

ISSUE 3: TensorFlow Installation Issues
----------------------------------------
Solution:
    pip install tensorflow==2.15.0

ISSUE 4: Script Execution Error
--------------------------------
Solution:
    - Ensure all scripts are in the same folder
    - Run scripts in order (1 through 7)
    - Check Python version: python --version

ISSUE 5: Memory Error
----------------------
Solution:
    - Close other applications
    - Restart Python/terminal
    - Run scripts individually instead of run_all.py
